## 🧑‍💻 About Me

I am a passionate Full Stack Web Developer with hands-on experience in developing modern, scalable, and user-centric web applications using the JavaScript and Python ecosystems.

I specialize in the **MERN** stack but have evolved into a **Polyglot Developer**, mastering **TypeScript** for type safety and **PostgreSQL** for robust relational data management. I am proficient in using **Prisma ORM** to bridge the gap between databases and applications efficiently.

On the frontend, I craft fast, responsive, and SEO-friendly interfaces using **React**, **Next.js**, and **Tailwind CSS**. On the backend, I build secure RESTful and **GraphQL** APIs using **Node.js**, **Express**, and **Python Django REST Framework**. I am also exploring **React Native** to build cross-platform mobile applications.

My goal is to deliver clean architecture, high performance, and maintainable code that solves real-world problems.
